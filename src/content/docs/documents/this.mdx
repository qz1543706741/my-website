---
title: this 指向
---

谈`this`指向，先谈函数执行上下文，`this`存在于执行上下文中，在函数调用时确定

## 函数调用方式

| 调用方式          | 函数中 `this`的指向                                         |
| ----------------- | ----------------------------------------------------------- |
| 全局调用          | 全局对象：global(node)、window(浏览器)、undefined(严格模式) |
| new               | 新对象 (bind 构造出来的函数，也满足该规则)                  |
| 对象调用          | 前面的对象                                                  |
| call、apply、bind | 第一个参数                                                  |

## 箭头函数

箭头函数的 `this`永远为闭合词法上下文的值
